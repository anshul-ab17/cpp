## Recognition Guide

Use HashMap when:
- Need O(1) lookup
- Frequency counting
- Index mapping
- Prefix sum counts
- Pair finding
- Grouping

Use HashSet when:
- Uniqueness
- Fast existence checks
- Duplicate detection
- Sequence problems
